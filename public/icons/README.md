# PWA Icons

This directory contains the icons for the Edupulse PWA.

## Required Icons

Generate PNG icons in the following sizes for the manifest.json:

- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

## Shortcut Icons (optional)

- interview-shortcut.png (96x96)
- resume-shortcut.png (96x96)
- jobs-shortcut.png (96x96)

## Generation Tools

Use online tools like:
- https://favicon.io/favicon-converter/
- https://www.favicon-generator.org/
- https://favicon.io/favicon-generator/

Or design tools like Figma, Adobe XD, etc.

## Icon Design Guidelines

- Use the Edupulse brand colors (#1976d2 primary, #dc004e secondary)
- Clean, modern design
- Scalable vector source recommended
- Transparent background preferred
